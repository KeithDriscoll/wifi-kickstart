# Wi‑Fi Kickstart 🚀

**Wi‑Fi Kickstart** is a Chrome extension designed to diagnose, monitor, and optimize your network connection with clarity and speed. Built for users who value precision, polish, and emotional resonance in their tools.

## ✨ Features

- **Speed Test Toggle** – Enable or disable built-in speed testing
- **WARP Status Check** – See if Cloudflare WARP is active
- **Network Score** – View a calculated score based on latency, jitter, and speed
- **Themes** – Switch between light, dark, or custom themes
- **Diagnostic Panel** – View IP, latency, jitter, and provider info
- **Modular UI** – Pill-style sliders for intuitive control
- **Support Links** – Quick access to homepage and support forum

## 🧠 Philosophy

Wi‑Fi Kickstart is built with a holistic QA mindset:
- **Technical robustness** meets **emotional clarity**
- Every toggle, divider, and animation is intentional
- Designed for users who care about both function and feel
