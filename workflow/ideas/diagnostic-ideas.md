# 🔍 Diagnostic Ideas for Wi-Fi Kickstart

Features to deepen diagnostic clarity and empower users.

- [ ] Detect captive portals and offer branded guidance (“Looks like you’re stuck behind a login wall”)
- [ ] Auto-retest after 30 seconds if failure detected
- [ ] Flag inconsistent DNS responses
- [ ] Add latency and jitter metrics
- [ ] Detect MAC address filtering or blocked devices
- [ ] Offer “Why is this happening?” links for each error type
- [ ] Add diagnostic presets (e.g. “Coffee shop mode”, “Home router check”)
